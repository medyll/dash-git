# S3-02: README Viewer

**Size:** M | **Sprint:** 3 | **Deps:** S3-01

## Description
Auto-detect and render README.md with syntax highlighting and GitHub-flavored markdown.

## Acceptance Criteria
- [ ] Auto-detect README.md, README.rst, or readme.txt
- [ ] Render markdown with syntax-highlighted code blocks
- [ ] Relative links in README resolve to repo paths
- [ ] Images load from repo
- [ ] Table of contents from headings (optional)
