# S1-01: SvelteKit Project Setup

**Size:** M | **Sprint:** 1 | **Deps:** none

## Description
Bootstrap the SvelteKit project in SPA mode with @medyll/css-base, configure adapter-static, and create the routing shell.

## Acceptance Criteria
- [ ] `npm create svelte` with Svelte 5, TypeScript
- [ ] `adapter-static` configured (SPA fallback enabled)
- [ ] `@medyll/css-base` installed and imported globally
- [ ] Base `+layout.svelte` renders a 3-panel shell (sidebar slot, tab bar slot, content slot)
- [ ] Hash-based routing works (`/#/` loads without 404)
- [ ] `npm run dev` starts without errors
- [ ] `npm run build` produces static output
