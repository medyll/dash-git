# S3-01: Repository Tree View

**Size:** L | **Sprint:** 3 | **Deps:** S2-01, S1-04

## Description
Build a file tree component showing repository structure with infinite nesting and lazy loading.

## Acceptance Criteria
- [ ] `repo_browser/TreeView.svelte` shows hierarchical file structure
- [ ] Lazy load directories on expand
- [ ] File icons by type (folder, file, markdown, code)
- [ ] Click file opens in new tab
- [ ] Search-in-tree: filter nodes by name
- [ ] Breadcrumb navigation showing current path
- [ ] Branch selector dropdown
