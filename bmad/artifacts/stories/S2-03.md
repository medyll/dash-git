# S2-03: Tab Bar UI Component

**Size:** M | **Sprint:** 2 | **Deps:** S2-02

## Description
Build the tab bar component showing open tabs with scroll, close, and switch functionality.

## Acceptance Criteria
- [ ] Horizontal scrollable tab bar
- [ ] Active tab highlighted
- [ ] Close button on hover (or always on mobile)
- [ ] Click tab to switch
- [ ] Auto-scroll active tab into view
- [ ] Tab shows: icon (by type), title, close button
- [ ] Overflow menu when too many tabs
