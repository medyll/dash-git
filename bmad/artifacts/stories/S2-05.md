# S2-05: Workspace Switcher

**Size:** S | **Sprint:** 2 | **Deps:** S2-01

## Description
Add a workspace switcher at top of sidebar to filter repos by owner/org.

## Acceptance Criteria
- [ ] Dropdown at top of sidebar showing "All repos", user, and orgs
- [ ] Filter repos by selected workspace
- [ ] Persist selection to localStorage
- [ ] Show org avatars in dropdown
- [ ] Keyboard shortcut to cycle workspaces (Ctrl+Shift+W)
