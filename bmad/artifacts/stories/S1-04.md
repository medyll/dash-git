# S1-04: GitHub GraphQL Client

**Size:** M | **Sprint:** 1 | **Deps:** S1-02

## Description
Create `github_client.ts` — a typed GraphQL transport that injects auth headers and handles errors.

## Acceptance Criteria
- [ ] `query<T>(document, variables): Promise<T>` function exported
- [ ] Auth token injected automatically from `auth_engine`
- [ ] Retry on 502/503 (max 2 retries, exponential backoff)
- [ ] Rate limit detection via `X-RateLimit-Remaining` header
- [ ] Request deduplication: identical in-flight queries return same Promise
- [ ] TypeScript types for GraphQL responses
