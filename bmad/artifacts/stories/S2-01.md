# S2-01: Global Sidebar Component

**Size:** L | **Sprint:** 2 | **Deps:** S1-03, S1-04

## Description
Build the global sidebar showing repository list with PR/CI status badges. Uses GitHub GraphQL API to fetch user's repos.

## Acceptance Criteria
- [ ] `global_sidebar.svelte` shows list of user's repositories
- [ ] Each repo shows name, primary language, and last update
- [ ] PR badge: shows count of open PRs (glowing indicator)
- [ ] CI badge: shows build status (green/yellow/red dot)
- [ ] Clicking repo opens it in a new tab
- [ ] Fuzzy search filter at top of sidebar
- [ ] Loading skeleton while fetching
- [ ] Error state with retry button
