# S2-04: Repo Status Badges

**Size:** M | **Sprint:** 2 | **Deps:** S2-01, S1-04

## Description
Create reusable badge components for PR count, CI status, and notifications.

## Acceptance Criteria
- [ ] `PRBadge.svelte` — shows count, glows when > 0
- [ ] `CIStatusBadge.svelte` — green/yellow/red/gray states
- [ ] `NotificationBadge.svelte` — dot indicator
- [ ] All badges use css-base colors
- [ ] Tooltips on hover explaining status
- [ ] Accessible: aria-labels for screen readers
