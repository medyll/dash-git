# S2-02: Tab Engine Core

**Size:** L | **Sprint:** 2 | **Deps:** S1-03, S1-05

## Description
Implement `tab_engine.svelte.ts` — reactive tab management with URL synchronization and localStorage persistence.

## Acceptance Criteria
- [ ] `Tab` interface: id, type, title, path, repo, isActive
- [ ] `tabs` reactive array with Svelte 5 runes
- [ ] `openTab(tab)`, `closeTab(id)`, `switchTab(id)` exported
- [ ] URL reflects active tab (hash-based routing)
- [ ] Tabs persist to localStorage on change
- [ ] Restore tabs on page load from URL + localStorage
- [ ] Max 20 tabs — oldest closed when exceeded
